= Building and deploying a RESTful application on a provisioned server

You can package the application with a provisioned server.

include::fragments/frag_application-home-info.adoc[]

.Prerequisites

* You have created an application.
+
For more information, see xref:creating-a-hello-world-rest-service_developing-a-hello-world-restful-service-application[Creating a Hello World REST service].

include::fragments/frag_building-and-deploying-to-provisioned-server-common.adoc[]

.Verification

. Navigate to the directory containing the provisioned server.
+
[source,options="nowrap"]
----
$ cd target/server
----

. Start the server.
+
[source,options="nowrap"]
----
$ ./bin/standalone.sh
----
+
In the log, you see that the application has been deployed as `ROOT.war`
+
----
...
21:05:35,275 INFO  [org.jboss.as.server] (ServerService Thread Pool -- 35) WFLYSRV0010: Deployed "ROOT.war" (runtime-name : "ROOT.war")
...
----

. To access the application, navigate to the address `http://localhost:8080/rest/HelloWorld`. You get the following message:
+
----
Hello World!
----
